# learn-claude-code

**Learn Claude Code by building TokyoSentences, a Japanese study tool.**

A hands-on course teaching you to build software with [Claude Code](https://www.anthropic.com/claude-code) ‚Äî Anthropic's AI coding assistant. No prior programming experience required.

---

## üéØ What You'll Build

**TokyoSentences** ‚Äî a desktop app for managing Japanese language study materials. By the end of this course, you'll have built:

- A native desktop application (Electron)
- A local database (SQLite)
- Full CRUD operations (create, read, update, delete)
- Version history and data export
- Search and filtering

More importantly, you'll learn how to describe what you want and let Claude Code build it.

---

## üß† The Core Skill

This course teaches one thing: **how to collaborate with AI to build software.**

You won't memorize syntax. You won't copy-paste from tutorials. Instead, you'll:

1. Describe what you want in plain English
2. Review what Claude Code proposes
3. Approve, refine, or redirect
4. Run and test the result
5. Repeat

This is how software gets built now.
